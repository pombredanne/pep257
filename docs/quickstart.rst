Quick Start
===========

1. Install

    .. code::

        pip install pep257

2. Run

    .. code::

        $ pep257 test.py
        test.py:18 in private nested class `meta`:
                D101: Docstring missing
        test.py:22 in public method `method`:
                D102: Docstring missing
        ...

3. Fix your code :)
